# IntelliJ Setup Guide

## Overview
This project was built on the **UC Berkeley CS61B Library** environment.  
It depends on the library-sp25 and runs with **Java 17**.

Setup:

---

## Prerequisites

- **IntelliJ IDEA (Community or Ultimate)**  

- **Java Development Kit (JDK)**  
  Recommended: **Amazon Corretto 17**


Go to File → Project Structure → Project.

Set:

Project SDK: Amazon Corretto 17 (or any JDK 17)

Project language level: SDK default (17 - Sealed types, always-strict floating point)

Click Apply → OK.
